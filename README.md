# 🎬 Movie Recommendation System

A hybrid movie recommendation system built with Python that suggests movies based on user preferences.

## 🔗 Live App
👉 https://movie-recommender-uyu3n9xvkzd4ltbaqu3hfq.streamlit.app/

## 📌 What it does
- Recommends movies based on what you have watched before
- Uses two different AI models for better accuracy
- Simple and easy to use web interface

## 🛠️ Built With
- Python
- Pandas
- Scikit-learn
- Surprise (SVD)
- Streamlit

## 📊 Dataset
MovieLens 100K Dataset — 100,000 ratings from 943 users on 1682 movies

## 👩‍💻 How it works
1. Content-Based Filtering — recommends movies similar to ones you liked
2. Collaborative Filtering — finds users similar to you and recommends their favourites
